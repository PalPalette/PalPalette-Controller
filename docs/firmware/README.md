# Firmware Binaries

This directory contains compiled firmware binaries for the PalPalette Controller.

## Files

- `manifest.json` - ESP Web Tools manifest file
- `info.json` - Build information (auto-generated by CI/CD)
- `PalPalette-Controller-ESP32C3-v*.bin` - Firmware for Seeed XIAO ESP32C3

## Automated Builds

Firmware binaries are automatically built and deployed by GitHub Actions when:

- Code is pushed to the `main` branch
- A version tag is created (e.g., `v2.0.0`)

## Manual Installation

If you prefer to flash manually using PlatformIO or esptool:

### Using PlatformIO

```bash
pio run -e seeed_xiao_esp32c3 -t upload
```

### Using esptool.py

```bash
esptool.py --chip esp32c3 --port COM3 write_flash 0x0 PalPalette-Controller-ESP32C3-v2.0.0.bin
```

Replace `COM3` with your device's serial port.

## Web Installer (Recommended)

For the easiest installation experience, use the web installer:
